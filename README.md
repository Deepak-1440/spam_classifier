# Spam Classifier (Machine Learning Project)

## Overview
This project is a Spam Message Classifier built using Machine Learning techniques. It predicts whether a given message is Spam or Ham (Not Spam) using Natural Language Processing (NLP).

The model is trained on labeled text data and can be used for real-time predictions through a simple interface.

---

## Features
- Classifies messages into Spam / Not Spam  
- Text preprocessing using NLP techniques  
- Feature extraction using vectorization  
- Machine learning model for prediction  
- Ready for deployment (Streamlit compatible)

---

## Tech Stack
- Python  
- Scikit-learn  
- Pandas  
- NumPy  
- Streamlit  
- Pickle
